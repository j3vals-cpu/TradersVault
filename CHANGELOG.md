Traders Vault v1.6.0 — Windows 10/11

## What's New

### Account & Authentication
- Users can now sign up and log in directly from the desktop app
- Fixed login validation — new accounts are activated instantly on first launch
- Automatic profile creation on signup (no manual admin step required)

### Platform & Backend
- Switched to custom domain: tradersvault.app
- Improved API routing and server stability
- Admin panel live at tradersvault.app/admin for user & licence management
- Automatic update checks — the app now detects when a newer version is available

### Core Features
- Trading panel with live TradingView overlay
- Prop firm tracker for challenge & funded account progress
- Position sizing calculator with risk management
- Bot engine for automated strategy execution
- Strategy & news feed panel
- Accountability journal & notes panel
- Weekly planner with session scheduling
- Gains card & studio tools for sharing results
- Discord integration for community alerts

### Desktop Experience
- OS-level transparent click-through overlay
- Multi-monitor support
- Dock bar mode for compact layout
- Portable .exe option (no install required)
