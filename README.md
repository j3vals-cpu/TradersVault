# Traders Vault — Build & Install Guide

Traders Vault is a transparent Windows desktop overlay app for traders.
Designed by **J3 | Traders Vault**

## Build steps

1. Install Node.js from https://nodejs.org (LTS version)
2. Open this folder in a terminal
3. Run: npm install
4. Run: npm start (to test)
5. Run: npm run build (to create the installer in dist/)

## How to use

- Click **⚓ Dock** to collapse into a slim dock bar at the top
- Enable Auto-hide to reveal dock on mouse hover at top edge
- Drag panels anywhere — including to a second monitor
- Click **⛴ Overlay** to float above TradingView
